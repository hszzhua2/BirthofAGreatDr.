# 概率实验室：Pluchino 人生成功随机性模拟器 🎲

> **“成功当然需要努力，但我们往往低估了命运在其中的权重。”**

这是一个基于意大利物理学家 **Alessandro Pluchino** 获“搞笑诺贝尔奖”的数学模型（Talent vs Luck）开发的交互式 Web 仿真应用。该实验室通过实时模拟 10,000 个体在 100 年间的财富演变，直观地揭示了**天赋、努力与机遇**之间残酷而迷人的博弈。

---

## 🏗 项目核心：Pluchino 模型逻辑

在该模拟器中，我们构建了一个简化的“微缩社会”：

1. **初始设定**：10,000 名代理人，初始资本（Success）均为 10。
2. **天赋分布**：每个人的天赋值（Talent）呈**正态分布**（μ=60, σ=15）。
3. **机遇事件（绿色）**：随机降临。若 $Roll(0-100) < Talent$，则个体抓住机遇，**财富翻倍 (x2)**。
4. **灾难事件（红色）**：随机降临。无法避开，**财富减半 (/2)**。
5. **结论分析**：通过 100 年的代际演化，观察谁成为了首富，而真正的“天才”又身处何方。

---

## ✨ 核心特性

* **高性能仿真引擎**：使用 `TypedArrays`（Float32Array/Int32Array）处理万级个体并行计算，配合 `requestAnimationFrame` 实现 60FPS 的流畅视觉演进。
* **可视化数据面板**：
* **社会样本网格**：10,000 个像素点实时变换色彩，直观展现财富两极分化。
* **对数刻度图表**：使用 Log10 坐标系展示财富长尾分布，动态呈现帕累托法则（20/80 原则）。


* **实时结论追踪**：
* **帕累托指标**：实时计算 Top 20% 人群掌握的财富占比。
* **首富回溯**：对比全场首富与全场天赋第一人的属性，拆解成功的偶然性。


* **极致视觉体验**：
* **Glassmorphism（玻璃拟态）** UI 设计，营造专业实验室质感。
* **响应式流体背景**，完美适配移动端与超宽屏显示器。

## 📊 实验观察笔记

在多次模拟实验中，你会发现：

* **马太效应**：财富分配最终总会符合帕累托法则，20% 的人掌握 80% 甚至更多的财富。
* **天才的宿命**：天赋最高的人（Talent > 95）如果没有遇到足够的绿色机遇，其最终财富往往仅处于社会平均水平。
* **幸运的平庸者**：首富往往是那些天赋中等偏上（如 70-80 分），但一生中从未遭遇重大灾难且捕捉到连续机遇的“幸运儿”。
